Slick-Pay.com Payment Gateway Plug-in for WooCommerce.

== Description ==

WordPress plugin for Slick-Pay API implementation, once installed, it provides a new payment gateway to the WooCommerce payment methods.

* Security, make secure transfers;
* Reliability, a reliable solution for your transfers;
* Mobility, by everyone at any time;
* Simplicity, make your payment with few clicks.

**One of the most important benefits of using Slick-Pay is that the transactions fees can be on the Slick-Pay account owner or the final client during the payment process.**

**[ New ]**: New feature implemented, now you are able to make your client pay only a **deposit** during the checkout.

Slick-Pay is available for Store Owners and Merchants in:

* Agleria

== Frequently Asked Questions ==

= Does this require an SSL certificate? =

Yes! A SSL certificate must be installed on your website to use the Slick-Pay gateway.

= How can I do tests? =

From the plugin settings you can select the API environment either Live (Production) or Sandbox (Test).

= Is there any support for this service? =

If you get any issue, feel free to [contact us](https://slick-pay.com/contact).
